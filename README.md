# LyricsConnect 🎵

A modern platform connecting **lyricists** and **singers** to collaborate on music production.

## Features

- ✍️ **Share Lyrics** - Lyricists post their song texts
- 🎤 **Find & Collaborate** - Singers discover and request lyrics
- 📤 **Share Audio** - Exchange vocal recordings
- 💬 **Direct Messaging** - Communicate with collaborators
- ⭐ **Ratings & Reviews** - Build your reputation

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Design:** Modern, Responsive, Mobile-Friendly
- **Future:** Node.js + MongoDB Backend

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Create an account as a Lyricist or Singer
4. Start collaborating!
